#  DisMesMots

A web application designed to help speech and language therapists manage aphasia rehabilitation sessions more efficiently.

## Features

- Patient management
- Exercise creation and customization
- Exercise templates
- Session planning
- Therapy follow-up
- Resource library (words, images, audio resources)
- Real-time exercise adaptation during sessions
- Centralized patient and session information
- Progress tracking

## Tech Stack

- React
- TypeScript / JavaScript
- HTML5
- CSS3

## About the Project

DisMesMots was developed as a team project in collaboration with speech and language therapists.

The project aims to address a real-world challenge: therapists often rely on paper-based materials and multiple disconnected tools to prepare and conduct aphasia rehabilitation sessions.

DisMesMots centralizes patient information, exercises, and session management into a single platform, making therapy preparation more efficient and allowing therapists to better personalize patient care.
